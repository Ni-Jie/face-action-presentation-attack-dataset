face action presentation attack (FAPA) dataset

命名为

志愿者id——视频id——类别id

AB_CD_E.mp4

志愿者id，从00开始，依次+1

视频id，C表示视频顺序，0-4，每人五个视频，D表示嘴巴还是眼睛的动作，0-嘴巴，1-眼睛

类别id，E表示属于什么类别，0-live，1-spoof1（有边框），2-spoof2（低亮度），3-spoof3（高亮度），4-spoof4（外界可移动光源）
